# Estructura de Datos

Este repositorio contiene el registro completo de la asignatura **Estructura de Datos**, incluyendo ejercicios prácticos, actividades evaluables y material de apoyo desarrollado a lo largo del curso.

## Contenido
- Implementaciones de estructuras de datos
- Ejercicios propuestos en clase
- Actividades y trabajos entregables
- Prácticas orientadas al análisis y uso eficiente de estructuras

## Estructuras abordadas
- Arreglos y listas
- Pilas y colas
- Listas enlazadas
- Árboles
- Grafos
- Algoritmos de búsqueda y ordenamiento

## Objetivo
Consolidar los fundamentos teóricos y prácticos de las estructuras de datos, desarrollando soluciones eficientes y bien estructuradas.

## Notas
El código está organizado por temas o unidades, con fines académicos y de seguimiento del progreso durante la asignatura.
